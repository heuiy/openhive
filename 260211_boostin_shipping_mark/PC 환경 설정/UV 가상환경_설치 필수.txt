# UV를 추천하는 이유

- 콘다는 의존성이 관리가 쉽지 않음
    - pip 로 설치했다가 의존성이 엉망이 되어서 힘들게 작업했던 프로젝트가 안 돌아가는 경우도 있음
- poetry
    - 의존성 관리가 엄격함. dependency 안 맞으면 설치 자체가 안됨
    - 근데 poetry 는 속도가 좀 느림
- uv 는 poetry 의 의존성 장점과 빠른 속도까지 잡았음
